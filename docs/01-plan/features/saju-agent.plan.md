# [Plan] 사주 AI 에이전트 서비스

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 전통 사주 풀이는 전문가에게만 의존해야 하고 비용이 높아 일반인 접근이 어렵다 |
| **Solution** | 만세력 알고리즘으로 정확한 사주팔자를 계산하고 Claude AI가 자연어로 상세 해설하는 웹 서비스 |
| **Functional UX Effect** | 이름/생년월일/시간 입력 → 사주팔자 + 오행분석 + AI 풀이 + 궁합 + 오늘운세 즉시 제공 |
| **Core Value** | 누구나 무료로 전문가 수준의 사주 분석을 받을 수 있는 공개 서비스 |

## Context Anchor

| 항목 | 내용 |
|------|------|
| **WHY** | 전통 사주 지식을 디지털화하여 AI와 결합, 대중이 쉽게 접근 가능하게 |
| **WHO** | 사주에 관심 있는 일반 대중 (20-50대), 한국어 사용자 |
| **RISK** | 만세력 알고리즘 정확도, Claude API 비용, 사주 해석의 주관성 |
| **SUCCESS** | 정확한 사주팔자 계산 + 상세한 AI 풀이 + 공유 가능한 결과 페이지 |
| **SCOPE** | 사주 계산/풀이 + 궁합 + 오늘운세 + PDF 공유, Next.js + Vercel + Claude API |

---

## 1. 요구사항

### 1.1 핵심 기능 (Must)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 사주팔자 계산 | 이름, 생년월일(양력/음력), 태어난 시간 입력 → 년주/월주/일주/시주 계산 |
| F-02 | AI 사주 풀이 | Claude API로 사주팔자, 오행, 십신 분석을 자연어로 상세 해설 |
| F-03 | 오행 분석 | 목/화/토/금/수 균형 시각화 |
| F-04 | 궁합 분석 | 두 사람의 사주 비교 및 AI 궁합 풀이 |
| F-05 | 오늘의 운세 | 현재 날짜 기준 일진/세운 분석 |
| F-06 | PDF 저장/공유 | 사주 결과를 PDF로 다운로드 또는 링크 공유 |

### 1.2 사주 계산 범위

- **입력**: 이름(선택), 성별, 생년월일(양력/음력 선택), 태어난 시간 (시간 모를 경우 선택 가능)
- **계산**: 천간지지 (년/월/일/시), 오행, 음양, 십신, 대운 시작 나이
- **출력**: 사주원국 + 대운표 + AI 해설

### 1.3 입력 정보

| 항목 | 필수 | 비고 |
|------|------|------|
| 이름 | 선택 | 결과 페이지 표시용 |
| 성별 | 필수 | 대운 방향 결정 |
| 생년월일 | 필수 | 양력/음력 선택 |
| 태어난 시간 | 선택 | 12지시 선택 (모르면 '모름' 선택) |

---

## 2. 기술 스택

| 항목 | 기술 | 이유 |
|------|------|------|
| Framework | Next.js 14 (App Router) | Vercel 최적화, SSR/SSG 지원 |
| Deployment | Vercel | 무료 시작, 글로벌 CDN, 자동 배포 |
| AI | Claude API (claude-sonnet-4-6) | 고품질 한국어 사주 풀이 |
| Styling | Tailwind CSS + shadcn/ui | 빠른 UI 개발 |
| PDF | html2canvas + jsPDF | 브라우저 사이드 PDF 생성 |
| 만세력 | 직접 구현 (TypeScript) | 정확한 한국식 사주 계산 |

---

## 3. 만세력 알고리즘 범위

- 60갑자 천간지지 매핑
- 년주: 입춘 기준 계산
- 월주: 절기 기준 (24절기)
- 일주: 율리우스 날짜 기반 갑자 계산
- 시주: 12지시 매핑
- 오행: 천간/지지별 오행 속성
- 십신: 일간 기준 상대적 관계
